Install the library

Requirements:

  • PHP 5.3 or higher
  • Curl with SSL support
  • The JSON PHP extension

Installation

  1. Using composer:
    The preferred method is via composer. Follow the installation instructions if you do not already have composer installed.
  2. Download Release from GitHub: Go to our GitHub releases tab and download the zip file that is included in the release.
